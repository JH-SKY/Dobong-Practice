# 🏃 사용자 시나리오 (User Journey)

1. **강사 탐색**: A씨는 'AI' 과목을 가르치는 선생님(`teacher`)을 검색하고, 맘에 드는 선생님을 찜(`like_teacher`)합니다.
2. **강좌 등록**: 선생님이 개설한 'RAG 챗봇 만들기' 강좌(`course`)를 확인하고 수강 신청(`listening_course`)을 완료합니다.
3. **학습 시작**: 강좌 안에 포함된 여러 개의 강의 영상(`lecture`)을 하나씩 시청하며 공부합니다. 너무 도움이 된 영상은 따로 찜(`like_lecture`)해둡니다.
4. **소통 및 질문**: 공부하다 궁금한 점은 자유게시판(`post`)에 올리고, 다른 친구들이 달아준 댓글(`comment`)을 확인합니다.
5. **성장 기록**: 완강 후에는 후기(`review`)를 남깁니다. 나중에 들어올 후배들은 AI가 요약해 준 A씨의 후기를 보고 강의를 선택하게 됩니다.
